## Obfuscation Patterns
The plugins use several obfuscation techniques:

1. **Package Names**: `g/r/r/p000/t/q/i/u/o/e/s` = `squire/plugins/internal`
2. **String Encryption**: XOR + Base64, AES/ECB, DES/ECB
3. **Dead Code**: `"".length();`, always-false conditionals
4. **Variable Names**: `lllIIIlllIII` (l/I combinations)
5. **Expression Obfuscation**: `" ".length()` = 1
